===========================================
GitHub Release 생성 방법
===========================================

1. 브라우저에서 이 URL로 이동:
   https://github.com/randyjlee/designdiff/releases/new?tag=v1.0.2

2. Title 필드에 입력:
   Release v1.0.2

3. Description 필드에 아래 내용을 복사해서 붙여넣기:
   (클립보드에 이미 복사되어 있습니다 - Command+V로 붙여넣기)

4. "Publish release" 버튼 클릭

완료!
